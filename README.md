# Data Access (PostgreSQL)

This project uses a PostgreSQL database that is **not publicly accessible by default**.

## For graders / reviewers
To review the application, use the deployed Dash web app (provided in the submission / README).

If you need direct database access for verification beyond what the UI exposes, please request it from the project author. Access may be provided via **time-limited, least-privilege credentials**.

## Notes
- No database credentials, connection strings, or dumps are included in this repository.
- The application reads data through parameterized SQL queries and restricted server-side configuration.
